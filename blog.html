1. What is DOM and what is the purpose of DOM?

DOM is a way to represent the webpage in a structured hierarchical way so that it will become easier for programmers and users to glide through the document.The Document Object Model (DOM) is a tool for web developers that allows them to create dynamic, interactive web pages that respond to user input and data changes in real time.Everything in webpage work as a object.With dom different type of activities happen in webpage like on button click go to another page.Facebook Like option is one kind of dom manipulation.

Purpose:

To giving user a good experience webpage should have interactive system. For adding interactivity to a button developers need DOM.To create dynamic webpage from static HTML webpage developers use DOM.
